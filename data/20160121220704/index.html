問題：映画の個別ページに表示されている映画の「作品画像のURL」を取得しましょう
作業ファイル：scraping_image.rb

  require 'mechanize'

解説

使用するウェブサイトのリンクURLは各自開いた映画の個別ページのものを使用してください。ここでは美女と野獣の個別ページを使用します。

scraping_image.rb

4
 
  require 'mechanize'

  agent = Mechanize.new
  page = agent.get("http://eiga.com/movie/80080/") # 映画の個別ページのHTML情報を取得

作品画像のURLはpictBoxクラスの中のimgタグ要素の属性値srcです。searchメソッドを使ってpictBoxクラスの中のimgタグ要素を取得します。

scraping_image.rb

  require 'mechanize'

  agent = Mechanize.new
  page = agent.get("http://eiga.com/movie/80080/")
  elements = page.search('.pictBox img') # pictBoxクラスの中のimgタグ要素

これでpictBoxクラスの中のimgタグ要素が取得できました。欲しいのはこのimgタグ要素の属性値srcなので繰り返し処理を行い、get_attributeメソッドで属性値を取得しましょう。

scraping_image.rb

  require 'mechanize'

  agent = Mechanize.new
  page = agent.get("http://eiga.com/movie/80080/")
  elements = page.search('.pictBox img')
  elements.each do |ele|
    puts ele.get_attribute('src') # get_attributeメソッドで属性srcの値を取得
  end

これでスクレイピングのプログラムができました。実行してみましょう。ターミナルで以下のような出力になったでしょうか。

ちゃんと取得できましたね。取得できた画像のURLを開いて確認してください。

これで完成なのですが、今回取得したいHTML要素は１つだけでした。このような場合searchメソッドのように該当するHTML要素を全て取得して、そのあとeachメソッドで繰り返すのは手間ですね。該当する要素を１つだけ取得したい場合はsearchメソッドの代わりにatメソッドを使います。

atメソッド

atメソッドはsearchメソッドと同じようにセレクタを指定して、該当するHTML要素を取得するメソッドです。ただし、searchメソッドのように該当するものが複数あってもすべて取得はせず、最初の１つしか取得しません。

atメソッドを使ってソースコードを書き直してみましょう。

ruby[1]{5,6}scraping_image.rb
1
2
3
4
5
6
 
  require 'mechanize'

  agent = Mechanize.new
  page = agent.get("http://eiga.com/movie/80080/")
  ele = page.at('.pictBox img') # searchからatに変更する
  puts ele.get_attribute('src')

5行目でsearchメソッドの代わりにatメソッドを使います。これでHTML要素のオブジェクトが１つだけ取得できます。１つなので繰り返し処理をする必要はなく、そのままget_attributeメソッドが使えます。

なお、カリキュラムに記載した通り、 ele.get_attribute('src') は、 ele[:src] と書くこともできます。より簡潔に書けるため、積極的に利用していきましょう。
